# 됐다끝dettakkt — 설치형 웹앱(PWA)

## 됐다끝이란?
됐다됐다끝! 이라고 외치는 저희 아기의 말버릇을 따서 만든, 개인용 할 일 앱입니다.
프레임워크·빌드 도구 없이 파일 몇 개로 동작합니다.
휴대폰 홈 화면에 설치할 수 있고, 오프라인에서도 열리며, 데이터는 사용하는 기기에 저장됩니다.

## 파일 구성
- `index.html` — 앱 본체
- `manifest.webmanifest` — 앱 이름·아이콘·색상 정보
- `sw.js` — 오프라인 캐시용 서비스워커
- `icon-192.png`, `icon-512.png`, `icon-maskable-512.png` — 홈 화면 아이콘

## 휴대폰에 설치
- **아이폰(Safari)**: 주소 접속 → 공유 버튼 → "홈 화면에 추가"
- **안드로이드(Chrome)**: 주소 접속 → 메뉴(⋮) → "앱 설치" 또는 "홈 화면에 추가"

설치해두면 아이콘으로 열리고, 저장 데이터도 더 안정적으로 유지됩니다.

## 저장에 대해
- 데이터는 **이 앱을 여는 그 기기·브라우저 안에만** 저장됩니다(서버 없음).
- 다른 기기나 다른 사람에게 자동으로 공유되지 않습니다. 각자 자기 기록을 씁니다.
- 브라우저의 "사이트 데이터 삭제"를 하거나 앱을 지우면 기록도 사라집니다.
- 여러 기기에서 같은 데이터를 쓰거나 친구와 공유하려면 별도의 서버/DB가 필요합니다.
